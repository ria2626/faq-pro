A simple and customizable FAQ plugin with custom post type, Gutenberg block, settings page, and toggle animation styles.

== Description ==
FAQ Plugin lets you create and display Frequently Asked Questions on your WordPress site with ease.

Features:
* Custom Post Type for FAQs
* Gutenberg block for easy insertion
* Shortcode support: [faq]
* Toggle animations: Slide or Fade
* FAQs grouped with simple HTML markup

== Installation ==
1. Upload the `faq-plugin` folder to the `/wp-content/plugins/` directory or use the WordPress Plugin Installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to FAQs → Add New to create your questions and answers.
4. Use the [faq] shortcode or the FAQ Gutenberg block to display them.

== Frequently Asked Questions ==
= How do I change the toggle animation style? =
Go to Settings → FAQ Settings and choose between Slide or Fade.

= Can I use this without Gutenberg? =
Yes! Just use the [faq] shortcode.
